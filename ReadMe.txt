How to play:
	- Download the sounds from the assignment in Sound Design 02826
	  from Samuel Chaput (s202250)
	- Launch the patch main.pd with Puredata extended available here : https://puredata.info/downloads/pd-extended
	- Launch the game via /builds/game.exe

Frequents issues :
	- If there are errors in the Unity editor on the first opening
	  of the project, close unity and try to restart it. It should
	  fix the errors.

Unity version : 2020.1.13
